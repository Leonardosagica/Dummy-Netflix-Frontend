Hello, world!

This is a Dummy Netflix for Frontend project practice purpose.

This project was boostrapped with Create React App, using Redux.

## structure
It Was designed with HTML and CSS vanilla, along with Javascript to bring functionalities.

## API
It was used mainly axios (mixed with fetchs inside)
The API used was a Public IMDB API, to bring movies banners, info, rate to illustrate my website.


## VIEW THE PROJECT WEBSITE
To start you need to install npm depedencies:
npm install

Next, You need to select specific paste (sorry for the mess hahaha): 
cd netflix-build-youtube

Finally, You can start the aplication:
npm start
